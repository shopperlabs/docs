title: Controllers
description: "Create custom controllers and Livewire pages for your Shopper admin panel."
---

Shopper's admin panel is built with Livewire components, but you can also add traditional controllers for custom pages. Custom page components are configured in `config/shopper/admin.php` under the `pages` key.

## Page Namespace

The `pages` configuration controls where Shopper looks for custom Livewire page components:

```php
// config/shopper/admin.php

'pages' => [
    'namespace' => 'App\\Livewire\\Shopper',
    'view_path' => resource_path('views/livewire/shopper'),
],
```

This means custom Livewire components placed in `app/Livewire/Shopper/` will be available in the admin panel. You can change this namespace to match your project structure.

## Custom Routes

To add custom routes that share Shopper's authentication middleware, point to a dedicated route file in `config/shopper/routes.php`:

```php
// config/shopper/routes.php

'custom_file' => base_path('routes/shopper.php'),
```

Routes defined in this file are automatically loaded with the admin panel's middleware stack. Create the route file and define your routes:

```php
// routes/shopper.php

use App\Livewire\Shopper\PostIndex;
use App\Livewire\Shopper\PostEdit;

Route::get('/posts', PostIndex::class)->name('shopper.posts.index');
Route::get('/posts/{post}', PostEdit::class)->name('shopper.posts.edit');
```

## Creating a Custom Page

Create a Livewire component in the configured namespace:

```bash
php artisan make:livewire Shopper/PostIndex
```

This creates `app/Livewire/Shopper/PostIndex.php` and `resources/views/livewire/shopper/post-index.blade.php`.

Your component can use Shopper's layout to match the admin panel styling:

```php
namespace App\Livewire\Shopper;

use Illuminate\Contracts\View\View;
use Livewire\Attributes\Layout;
use Livewire\Component;

#[Layout('shopper::components.layouts.app')]
class PostIndex extends Component
{
    public function render(): View
    {
        return view('livewire.shopper.post-index');
    }
}
```

<Tip>
Use `#[Layout('shopper::components.layouts.app')]` on your component to inherit the admin panel layout with sidebar, header, and notification system.
</Tip>

## Adding Navigation

To add your custom page to the admin sidebar, create a sidebar extension. See the [Navigation](/v2/extending/navigation) page for details on adding menu items.
