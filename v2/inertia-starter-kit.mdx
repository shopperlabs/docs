The Inertia Starter Kit provides a modern single-page application storefront using [Inertia.js](https://inertiajs.com/). It lets you use [React](https://react.dev/) or [Vue.js](https://vuejs.org/) for the frontend while keeping the full power of Laravel on the backend, with no separate API layer needed.

<Note>
This starter kit is currently in development. Check the [Shopper repository](https://github.com/shopperlabs/shopper) for updates on the release timeline.
</Note>

In the meantime, you can get started with the [Livewire Starter Kit](/v2/livewire-starter-kit), which is available today.
